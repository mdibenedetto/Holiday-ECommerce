# git

## 1 Stage changes

- a: Add your files one by one
  `git add FILE_NAME`

- .b Add all files
  `git add .`

## 2. Commit
`git commit -m "HERE A MESSAGE WHICH EXPLAIN WHAT YOU HAVE DONE" `

## 3. Push changes to the server of github
`git push`

## 4. to pull down changes made by another developer
`git pull`
